Jan's ECE Coursework

Projects

1. Nim

Course: EE306 (Fall 2019)

Description: The game board consists of three rows of rocks. Row A contains 3 rocks, Row B contains 5 rocks, and Row C contains 8 rocks. Each player takes turns removing one or more rocks from a single row. A player cannot remove rocks from more than one row in a single turn. The game ends when a player removes the last rock from the game board. The player who removes the last rock loses.

Materials: LC3, Assembly concepts, and Nim rules

2. Electrical Etch-A-Sketch

Course: EE319K (Spring 2020)

Description: Implement an Etch-A-Sketch board on an embedded system. The board's x- and y-axes are controlled by two potentiometers, and the color of the line can be changed with the press of a button.

Materials: Tiva C Micro Controller, potentiometers, 1 button, 1 breadboard, and 1 LCD screen

3. Cheaters

Course: EE312 (Fall 2020)

Description: Check for plagarism in essays submitted by students. The algorithm is simple and consists of: Pre-processing the text, take a command input, compare the list from the input provided, repeat the steps for the other essays, sort the pairs in decending order, and take input variable m to print the sorted pairs.

Materials: CLion, C++, and a text file of previously written essays.

4. Word Ladder

Course: EE422C (Spring 2021)

Description: Create and output a word ladder from a given start and end word. The ladder must be the shortest possible and each word must be a valid word from the dictionary file given.

Materials: Eclipse, Java, and a text file of words as the dictionary.

5. Bidding Server-Client Network

Course: EE422C (Spring 2021)

Description: This project demonstrates a server-client architecture. The goal is to design and implement an online auction server and client backend. All communication between the server and clients will occur through sockets, therefore the program will work even if clients are not run on same machine.

Materials: Eclipse, Java, JavaFX, and socket programming library.
